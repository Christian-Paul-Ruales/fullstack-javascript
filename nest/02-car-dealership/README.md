# DOCUMENTACION OFICIAL

## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```

# DOCUMENTACION DEL CURSO

## iniciar el proyecto
```bash
yarn start:dev
```

## Descripcion de archivos del proyecto

* lintr (eslintrc.js): buenas practicas siguiendo ciertas reglas
* prettierrc: seguir buenas practicas de la mano lintr, pero mas esteticas, reglas de espacios por ejemplo
* package json maneja dependencias y scripts
* tsconfig.build.json: informacion de versiones de produccion, usa el tsconfig.json
* tsconfig.json: reglas para typescript, para manejar codigo limpio

## carpetas
* dist: producto final de la aplicacion
* node_modules: dependencias descargadas
* src: carpeta de codigo del prouecto
* test: archivos de prueba
